# Data-Science-AUC-Metric-Evaluation

## _Project Overview_

Berikut adalah data tentang class Gaji seseorang. Data ini memilki beberapa column predictor dan 1 column output untuk 2 class label, Anda di minta untuk membuat model untuk memprediksi gaji seseorang, dengan ketentuan sebagai berikut:

0 : Gaji <= 6 juta 1 : Gaji > 6 juta

Silahkan download file train.csv sebagai data training, dan gunakan data test.csv sebagai data test

## _Goals_
Evaluasi matriks yang di gunakan adalah AUC, melakukan scoring metrics roc_auc terhadap model yang dibuat.
